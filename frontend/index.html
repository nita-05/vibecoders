FAQ

Does it generate multiple scripts? +
Yes. If you ask for multiple systems, include placement headers like -- Script: ServerScriptService/GameManager.
Is the code safe to paste into Studio? +
The generator focuses on minimal, complete scripts. There’s also a light “review first” signal if outputs look risky, but you should still review anything that gets placed into your game.
Do I need images? +
No. Text-only prompts work great. Images are optional and can help steer the model (up to 2 images).
Where do I start? +
Open the builder, paste a starter prompt, and iterate: refine the prompt after you test in Studio.
